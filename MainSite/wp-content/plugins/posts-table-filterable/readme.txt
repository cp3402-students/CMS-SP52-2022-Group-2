TABLEON - Posts Table Filterable: WordPress plugin for displaying and filter posts and their custom post types in table format.

== Description ==

**WordPress Posts Table Filterable** - (TABLEON) - WordPress plugin for displaying and filter wordpress posts and their custom post types in table format. WP Tables makes focus for your customers on the things they want to get, nothing superfluous, just what the client wants, and full attention to what is offered!

Latest PHP 8.x - FULL COMPATIBILITY!

This plugin is FREE, no limitations in features, but can be extended by [free and premium extensions](https://pluginus.net/product-category/tableon/).

Demo site: [demo.posts-table.com](https://demo.posts-table.com/).


### WP Posts Table Features:

&#9989;&nbsp;Columns constructor

&#9989;&nbsp;Options for: columns customizations, thumbnail size, columns width and many more (screens below)

&#9989;&nbsp;Neat shortcode [[tableon]](https://posts-table.com/shortcode/tableon/) with heap of attributes for flexibility

&#9989;&nbsp;More than 10 predefined column fields is possible to display

&#9989;&nbsp;[Possibility to create custom columns](https://posts-table.com/how-to-add-custom-column-to-the-tables/)

&#9989;&nbsp;Skins for each table. On the same web page can be represented [some posts tables with the different skins](https://demo.posts-table.com/different-skins/). Create [your own skins by CSS](https://posts-table.com/document/skins/).

&#9989;&nbsp;Powerful posts filter constructor

&#9989;&nbsp;[Big variety of filter elements](https://demo.posts-table.com/tableon-with-all-possible-filters/) by: post_title, content, excerpt, category, tags, taxonomy, meta fields, etc...

&#9989;&nbsp;Filters can be represented as: dropdown, multi drop-down, range sliders, textinput, calendars

&#9989;&nbsp;Predefinition mechanism: show to your customers predefined and relevant set of posts as in the table, so in the popup.

&#9989;&nbsp;More than 10 ways of the posts sorting

&#9989;&nbsp;Shortcode [[tableon_drop_down]](https://posts-table.com/shortcode/tableon_drop_down/) allows to show the table below on the textinput when user searching the posts by its title. Place it on the shop header and let your customers find the posts they want to buy

&#9989;&nbsp;Power feature as [remote tables](https://inbuilt.posts-table.com/) which gives ability to spread your shop posts to any another sites even created on pure HTML without any CMS, what allows you get more sells and also realize your own referral program

&#9989;&nbsp;Column "Gallery" - smooth CSS gallery to showcase the best of your posts. Also a post gallery can be represented by shortcode [tableon_gallery](https://posts-table.com/shortcode/tableon_gallery/)

&#9989;&nbsp;[FREE ext.](https://pluginus.net/shop/tableon/tableon-favourites/) Shortcode [[tableon_favourites]](https://posts-table.com/shortcode/tableon_favourites/) allows to show the table with selected posts by the current user

&#9989;&nbsp;[Premium ext.](https://pluginus.net/shop/tableon/tableon-attachments/) Shortcode [[tableon_attachments]](https://posts-table.com/shortcode/tableon_attachments/) allows to show the table with post attachment for the current post on its single page or any another one

&#9989;&nbsp;[Premium ext.](https://pluginus.net/shop/tableon/tableon-compare/) Shortcode [[tableon_compare]](https://posts-table.com/shortcode/tableon_compare/) allows to show the table with selected compared posts by the current user or predefined by shop admin

&#9989;&nbsp;Shortcode [tableon_single](https://posts-table.com/shortcode/tableon_single/) allows to show the table with a post parameters for any one selected ones

&#9989;&nbsp;Ability to use [Load More](https://demo.posts-table.com/table-with-load-more-button/) button instead of pagination

&#9989;&nbsp;Ability to show multi vendors their posts using attribute param author: [[tableon author=23]](https://posts-table.com/shortcode/tableon/)

&#9989;&nbsp;Ability to create custom columns, for example such as [Ask manager about the post](https://posts-table.com/shortcode/tableon_popup_iframe_button/) using any contact form plugins

&#9989;&nbsp;Responsivity: all the posts tables have an option for compact view on mobile devices (option: "Compact view width")!

&#9989;&nbsp;Big and flexible set of options for each post table, some of them: Show Sorting Dropdown, Sorting Dropdown Fields, Compact view width, Use load more button, Hide filter form, Show print button, Default order by, Per page drop-down position, Per page drop-down position, Per page values, Per page default

&#9989;&nbsp;Set of options for each post column, some basic: Width, Font size, Font family, Color, Background, Hide on small screen

&#9989;&nbsp;Ability to [create custom columns](https://posts-table.com/how-to-add-custom-column-to-the-tables/) specially for your business (by PHP code)

&#9989;&nbsp;Ability to add meta fields data in the table columns using in-built constructor

&#9989;&nbsp;Ability to [show a post table in the popup on click](https://demo.posts-table.com/table-as-link/). Show tables in popup in your shop text content with relevant predefined tables. Unlimited count of tables in popup per page can be created!

&#9989;&nbsp;[Caching table of posts](https://demo.posts-table.com/cached-table-of-posts/) mechanism

&#9989;&nbsp;Print button

&#9989;&nbsp;Mobile compact view option

&#9989;&nbsp;CSS editor

&#9989;&nbsp;Pagination

&#9989;&nbsp;[Compatible with WPCS](https://wordpress.currency-switcher.com/posts-table-tableon-is-compatible-with-wpcs/)

&#9989;&nbsp;[Compatible with WPBE](https://bulk-editor.pro/)

&#9989;&nbsp;[Compatible with MDTF](https://cars.wp-filter.com/tableon-compatibility/) (redirect mode)

&#9989;&nbsp;WPML

&#9989;&nbsp;[Custom Post Type UI](https://wordpress.org/plugins/custom-post-type-ui/)

&#9989;&nbsp;Works in Elementor and Page builder page content

&#9989;&nbsp;[Power and strong API](https://posts-table.com/codex/)

&#9989;&nbsp;no jQuery - 100% pure JavaScript

&#9989;&nbsp;✔ Strong technical support which each day works with tones of code!
